The Knowledge Gap Detection System is a Machine Learning–based application developed to analyze student performance and systematically identify learning deficiencies across multiple subject areas. The system utilizes K-Means clustering to group learners based on performance patterns, enabling meaningful comparisons between individual scores, cluster averages, and predefined target benchmarks. It performs a comprehensive gap analysis to detect weak areas and computes a weighted risk score to quantify the severity of knowledge gaps. Based on this analysis, learners are categorized into risk tiers such as low, moderate, high, and critical. Additionally, the system incorporates a confidence scoring mechanism to evaluate prediction reliability and supports data-driven decision-making. To enhance usability, an interactive dashboard built with Streamlit visualizes performance insights and provides a personalized intervention planner with actionable recommendations. This project demonstrates the effective application of machine learning in learning analytics, performance evaluation, and personalized education systems.
